# pipenv
run `pipenv` console command, in console add packages e.g. `pipenv isntall flask`


# run app
 - set required API KEYS:
   ```env
   OPENAI_API_KEY=
   PROXYCURL_API_KEY=
   PYTHONUNBUFFERED=
   SERPAPI_API_KEY=
   ```
 - consider changing of ice_break's `real_mode` flag to `True`
